## ⚙️ Installation
1. Install **Python 3.10+**
2. Clone the repository:
   ```bash
   git clone https://github.com/Rahulkhetwal/Smart-Project-Builder.git
   cd Smart-Project-Builder
   ```
3. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
